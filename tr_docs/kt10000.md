# 주식 매수주문 (kt10000) API 명세

## 1. 기본정보

- **API 명**: 주식 매수주문
- **API ID**: kt10000
- **Method**: POST
- **운영 도메인**: `https://api.kiwoom.com`
- **모의투자 도메인**: `https://mockapi.kiwoom.com` (KRX만 지원가능)
- **URL**: `/api/dostk/order`
- **Format**: JSON
- **Content-Type**: `application/json;charset=UTF-8`

---

## 2. Request (요청)

### Header

| 구분   | Element       | 한글명       | Type   | Required | Length | Description                                                                             |
| :----- | :------------ | :----------- | :----- | :------: | :----- | :-------------------------------------------------------------------------------------- |
| Header | api-id        | TR명         | String |    Y     | 10     | 7자리 TR코드, ex) kt10000                                                               |
| Header | authorization | 접근토큰     | String |    Y     | 1000   | 토큰 지정시 토큰타입("Bearer") 붙혀서 호출 예) Bearer Egicyx...                         |
| Header | cont-yn       | 연속조회여부 | String |    N     | 1      | 응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 cont-yn값 세팅  |
| Header | next-key      | 연속조회키   | String |    N     | 50     | 응답 Header의 연속조회여부값이 Y일 경우 다음데이터 요청시 응답 Header의 next-key값 세팅 |

### Body

| 구분 | Element      | 한글명         | Type   | Required | Length | Description                                                                                                                                                                                                                                                                             |
| :--- | :----------- | :------------- | :----- | :------: | :----- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Body | dmst_stex_tp | 국내거래소구분 | String |    Y     | 3      | KRX, NXT, SOR                                                                                                                                                                                                                                                                           |
| Body | stk_cd       | 종목코드       | String |    Y     | 20     | 종목코드 6자리                                                                                                                                                                                                                                                                          |
| Body | ord_qty      | 주문수량       | String |    Y     | 20     | 매수 주문수량 (단위: 1주)                                                                                                                                                                                                                                                               |
| Body | ord_uv       | 주문단가       | String |    Y     | 20     | 매수 주문단가 (단위: 원)                                                                                                                                                                                                                                                                |
| Body | trde_tp      | 매매구분       | String |    Y     | 2      | 0:보통 , 3:시장가 , 5:조건부지정가 , 81:장마감후시간외 , 61:장시작전시간외, 62:시간외단일가 , 6:최유리지정가 , 7:최우선지정가 , 10:보통(IOC) , 13:시장가(IOC) , 16:최유리(IOC) , 20:보통(FOK) , 23:시장가(FOK) , 26:최유리(FOK) , 28:스톱지정가,29:중간가,30:중간가(IOC),31:중간가(FOK) |
| Body | cond_uv      | 조건단가       | String |    N     | 20     | 조건단가                                                                                                                                                                                                                                                                                |

---

## 3. Response (응답)

### Header

| 구분   | Element  | 한글명       | Type   | Required | Length | Description                         |
| :----- | :------- | :----------- | :----- | :------: | :----- | :---------------------------------- |
| Header | api-id   | TR명         | String |    Y     | 10     | 7자리 TR코드, ex) kt10000           |
| Header | cont-yn  | 연속조회여부 | String |    N     | 1      | 다음 데이터가 있을시 Y값 전달       |
| Header | next-key | 연속조회키   | String |    N     | 50     | 다음 데이터가 있을시 다음 키값 전달 |

### Body

| 구분 | Element      | 한글명         | Type   | Required | Length | Description                                    |
| :--- | :----------- | :------------- | :----- | :------: | :----- | :--------------------------------------------- |
| Body | ord_no       | 주문번호       | String |    N     | 20     | 정상 접수 시 발급되는 주문번호                 |
| Body | dmst_stex_tp | 국내거래소구분 | String |    N     | 3      | 실제 주문이 처리된 거래소 구분 (KRX, NXT, SOR) |

---

## 4. Response Example (응답 예시)

```json
{
  "ord_no": "0000123",
  "dmst_stex_tp": "KRX",
  "return_code": 0,
  "return_msg": "정상적으로 처리되었습니다"
}
```
